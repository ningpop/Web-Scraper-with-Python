# Web Video Downloader with Python

## How to Use

1. 원하는 영상 소스 링크를 찾는다.<br/>
(크롬에서 F12 개발자 도구로 찾는다.)<br/>
2. downloader.py 내 12번째 줄 url변수에 링크를 넣는다.<br/>
3. 아래와 같이 실행한다.

- Windows
``` bash
$ python -m venv venv
$ source venv/Scripts/Activate
$ python downloader.py
```

- Mac OS
``` bash
$ python3 -m venv venv
$ source venv/bin/activate
$ python downloader.py
```
